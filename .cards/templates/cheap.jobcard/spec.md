# Cheap Template — Example Spec

> This template runs a single implement stage on a local model (Ollama).
> Use for small, well-defined tasks where speed and cost matter
> more than sophistication. No QA stage — acceptance criteria are the gate.

## Task
Describe a small, concrete change (one function, one fix, one file).

## Requirements
- Keep scope tiny — local models have ~4K context
- Be explicit — no ambiguity, no "figure it out"

## Acceptance Criteria
- [ ] Code compiles
- [ ] Tests pass
